# NASSAL Website

A modern nonprofit website for the **National Stroke Survivors Association of Liberia (NASSAL)**, built to promote **awareness, survivor support, prevention education, community outreach, and hope**.

## Overview

This project was created to give NASSAL a strong digital presence with a clean, trustworthy, and professional design. The website highlights the organization’s mission, leadership, projects, and contact information while making it easier for visitors, partners, donors, and community members to connect with the organization.

## Features

- Modern responsive design
- Multi-page layout
- Home, About Us, Projects, and Contact Us pages
- Rounded modern navigation
- Professional hero section
- Leadership / “Who We Are” section
- Projects and outreach showcase
- Contact form section
- Nonprofit / healthcare-focused UI
- Mobile, tablet, and desktop friendly

## Pages

### Home
Focuses on **trust, mission, and impact**.

### About Us
Highlights **the organization’s story, mission, values, and leadership**.

### Projects
Showcases **real work, outreach efforts, and community initiatives**.

### Contact Us
Provides **form, email, phone, and location details** for communication.

## Design Direction

The site was designed with a **modern nonprofit / healthcare aesthetic** using:

- Clean layouts
- Strong typography
- Soft shadows
- Rounded UI elements
- Warm and trustworthy visual tone
- Brand-inspired colors from the NASSAL logo

## Brand Inspiration

The design draws from the official NASSAL logo, using tones such as:

- Deep red
- Warm gold/orange
- White / light neutrals
- Charcoal / dark tones
- Subtle teal accents

## Purpose

The goal of this website is to help NASSAL:

- Raise awareness about stroke and stroke prevention
- Support stroke survivors and caregivers
- Present its mission professionally
- Showcase projects and outreach programs
- Improve public trust and visibility
- Make communication easier for partners, donors, and the public

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Framer Motion
